## [트리플] 일정 추천 알고리즘 구현

### 문제 정의
- 주어진 시간동안 어떤장소를 방문하야 할 지 미리 구해주는 서비스를 제공 하고자함.
- N개의 장소들의 리스트와 장소들을 방문했을 때 소요하는 시간, 각 장소를 방문했을 때 사용자가 얻는 만족도가 주어짐
- M시간 동안 여행하여 얻을 수 있는 최대 만족도를 구하는 프로그램 작성.
  

- 예제 
- |장소명|소요시간|만족도|
  |------|---|---|
  |오사카성|2|5|
  |유니버셜 스튜디오|5|20|
  |도톤보리|1|10|
  |기온|3|30|
- 이때 5시간 동안 얻을 수 있는 최대 만족도는 도톤보리와 기온을 방문했을 때의 만족도(10 + 30 = 40)이다.
- 3시간 동안 여행했을 때 얻을 수 있는 최대 만족도는, 기온만 방문 했을 때의 만족도 30이다.

#### 입력과 출력
- 모든 입력과 출력은 stdin, stdout으로 주어 짐을 가정.

#### 입력
- 첫번째 줄에는 장소들의 갯수(N <= 20)와 여행 시간(M <= 1000)이 주어짐
- 2번째 줄 부터 N + 1번째 줄까지는 각 장소들의 이름, 소요시간, 만족도가 ,로 구분되어 주어짐.

#### 출력
- 한 줄에 최대 만족도를 출력합니다.
